# MindOfMemory

This is a memory based game using react, useEffect and useState. 
You try to match 2 similar card with each other until you match them all.

## Installation

```bash
git clone https://github.com/FaysalShuvo/MindOfMemory.git
cd MindOfMemory/
npm install
npm start
```

### Technology used:

- react
- css
